Entra ID OIDC onboarding

identity

Inactive account audit tool (PowerShell)

identity, JML

Entra OIDC token test framework

identity, testing

A framework that tests OIDC token validation against a live Entra tenant (29 tests). Key finding: an unregistered redirect URI is not rejected up front. Entra shows the login page first and only blocks after authentication, and a mismatched token audience caused repeated 401 loops.

github.com/ddasfour/entra-oidc-token-test-framework
